# Basic concepts

[Maticon Office Docs](https://www.maticonoffice.ru/download-docs.aspx?from=api#docs-developer) is an open-source office suite that includes editors for documents, spreadsheets, presentations, PDFs, and fillable forms.

This documentation describes:

- How to extend the editors functionality with [plugins](../usage-api/config/editor/plugins.md) and [customization settings](../usage-api/config/editor/customization/customization-standard-branding.md), or limit it to suit your use-case
- How to manage editors via [Automation API](../usage-api/automation-api/automation-api.md)
- How to use the [WOPI protocol](../using-wopi/overview.md) to integrate your application with an online office
- How to integrate Maticon Office Docs into a range of [frameworks](./frontend-frameworks/react.md) and [Document Management Systems (DMSs)](./ready-to-use-connectors/nextcloud-integration.md)

## What editors can do

Maticon Office Docs provides the following features:

- creating, editing and viewing documents, spreadsheets, presentations, PDFs, and fillable forms;
- collaborating on files with other teammates in real time.

Maticon Office Docs API is used to let the developers integrate the Maticon Office Document/Spreadsheet/Presentation/Pdf Editors into their own web sites and setup and manage the editors. Maticon Office Docs can be customized to look and feel like part of your application or webpage by [customizing the user interface](../usage-api/config/editor/customization/customization-standard-branding.md).

## How to get editors

To get Maticon Office Docs and start using its features:

- load it from the [Maticon Office Docs Cloud CDN](./installation/cloud.md) (Content Delivery Network), which will ensure Maticon Office Docs is always using the latest version;
- install the [self-hosted](./installation/self-hosted.md) solution.
