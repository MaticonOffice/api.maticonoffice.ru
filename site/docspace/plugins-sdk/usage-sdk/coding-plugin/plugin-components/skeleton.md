# Skeleton

A component that is used to hide components during uploading.

**Interface**: [ISkeleton](https://github.com/MaticonOffice/docspace-plugin-sdk/blob/master/src/interfaces/components/ISkeleton.ts).

## Example

``` ts
import {type ISkeleton} from "@maticonoffice/docspace-plugin-sdk"

const skeletonProps: ISkeleton = {
  width: "100%",
  height: "100%",
}
```
